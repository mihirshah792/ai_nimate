## Overview
This project is an **MCP (Model Context Protocol) server** that enables LLMs (Claude) to generate **Manim-based animations** from user prompts and return a **ready-to-watch video URL**.

Unlike generic AI video tools, this system produces **deterministic, code-driven animations**, making it ideal for **computer science explanations, algorithms, and technical education**.
## Tech Stack

- **Typescript** : Building MCP server.
- **AWS** : Used AWS S3 bucket for video storage.
- **Claude** : Used as client LLM.


## Getting Started

Clone the project

```bash
  git clone https://github.com/mihirshah792/ai_nimate.git
```

Install dependencies

```bash
  npm install
```

Note: Do not forget to alter .env with your credentials.

Build the project

```bash
  npm run dev
```
Open Claude Desktop and navigate to Developer Settings, then edit the `claude_desktop_config.json` file as shown below.

```bash
{
  "mcpServers": {
    "animation-generator": {
      "command": "node",
      "args": [
        "/path/to/mcp_animation_generator/build/index.js",
        "--stdio"
      ],
      "timeout": 600000,
      "env": {
        "S3_BUCKET_NAME": "s3bucket_name",
        "AWS_REGION": "ap-south-1",
        "AWS_ACCESS_KEY_ID":"your_access_key_id",
        "AWS_SECRET_ACCESS_KEY":"your_access_key",
        "MANIM_PATH":"/path/to/mcp_animation_generator/venv/bin/manim",
        "VENV_BIN_PATH":"/path/to/mcp_animation_generator/venv/bin/",
        "VIRTUAL_ENV":"/path/to/mcp_animation_generator/code_compiler/venv"
      }
    }
  }
}
```

Restart the Claude Desktop application. After restarting, the exposed prompt and video rendering tool will appear in the *Add Files* section.
